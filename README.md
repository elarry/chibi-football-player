# Chibi Football Player

A chibi-style 3D football player character created in Blender that is rigged, animated, and exported to `.fbx` for use in Unity.

## Contents

| File | Description |
|------|-------------|
| `Chibi-Football-Player.blend` | Blender source file with rig and animations |
| `Chibi-Football-Player.fbx` | Exported model ready for Unity import |
| `assets/` | UV texture variants for the torso |

### Texture variants

- `UV-Torso-flower.png`
- `UV-Torso-Striker-black-knights.png`
- `UV-Torso-Striker-red-pixel.png`
